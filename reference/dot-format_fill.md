# Format a fill value as a character string

Format a fill value as a character string

## Usage

``` r
.format_fill(x)

# Default S3 method
.format_fill(x)

# S3 method for class 'Date'
.format_fill(x)
```

## Arguments

- x:

  A fill value.

## Value

(`character(1)`) An R expression string representing `x`.
