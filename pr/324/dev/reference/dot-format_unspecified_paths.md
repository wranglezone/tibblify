# Prep unspecified paths for messaging

Prep unspecified paths for messaging

## Usage

``` r
.format_unspecified_paths(path_list, path = character())
```

## Arguments

- path_list:

  (`list`) A list of
  [`tib_unspecified()`](https://tibblify.wrangle.zone/dev/reference/tib_spec.md)
  objects and objects containing
  [`tib_unspecified()`](https://tibblify.wrangle.zone/dev/reference/tib_spec.md)
  objects.

- path:

  (`character`) Current path prefix.

## Value

The formatted paths as `character`.
