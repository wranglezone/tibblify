# Map to a character vector

Map to a character vector

## Usage

``` r
.compat_map_chr(x, .f, ...)
```

## Arguments

- x:

  (`any`) The object to check.

- .f:

  (`function`) Function to apply to each element of `x`.

- ...:

  Additional arguments passed to `.f`.

## Value

(`character`) The mapped character vector.
