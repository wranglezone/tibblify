# Combine ptypes from a list of tibs

Combine ptypes from a list of tibs

## Usage

``` r
.tib_combine_ptype(tib_list, .call)
```

## Arguments

- tib_list:

  (`list`) A list of tib fields.

- .call:

  (`environment`) The environment to use for error messages.

## Value

(`vector(0)`) The combined ptype.
