# Parse a single media type object from an OpenAPI spec

Parse a single media type object from an OpenAPI spec

## Usage

``` r
.parse_media_type_object(media_type_object, openapi_spec)
```

## Arguments

- media_type_object:

  (`list`) A single media type object from an OpenAPI spec.

- openapi_spec:

  (`list`) A parsed OpenAPI specification.

## Value

A tibblify spec derived from the media type object's schema.
