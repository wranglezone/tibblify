# Wrap a string in double quotes if it's not `NULL`

Wrap a string in double quotes if it's not `NULL`

## Usage

``` r
.double_quote(x)
```

## Arguments

- x:

  (`character(1)` or `NULL`) A string to wrap in double quotes, or
  `NULL`.

## Value

The input string wrapped in double quotes if it was not `NULL`, or
`NULL`.
