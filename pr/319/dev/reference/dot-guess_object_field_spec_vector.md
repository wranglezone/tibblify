# Guess the field spec for a vector-typed field

Guess the field spec for a vector-typed field

## Usage

``` r
.guess_object_field_spec_vector(value, name)
```

## Arguments

- value:

  (`list`) An object list whose fields will be guessed.

- name:

  (`character(1)`) The name of the field.

## Value

A tib field specification.
