# Check that id column has no missing or duplicate values

Check that id column has no missing or duplicate values

## Usage

``` r
.check_id(x, x_arg, call = caller_env())
```

## Arguments

- x:

  (`any`) The object to check.

- x_arg:

  (`character(1)`) The name of the x argument. This name will be
  mentioned in error messages as the input that is at the origin of a
  problem.

- call:

  (`environment`) The environment to use for error messages.

## Value

The input `x`.
