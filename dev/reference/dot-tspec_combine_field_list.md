# Combine fields from a list of specs

Combine fields from a list of specs

## Usage

``` r
.tspec_combine_field_list(spec_list, .call)
```

## Arguments

- spec_list:

  (`list`) A list of specifications.

- .call:

  (`environment`) The environment to use for error messages.

## Value

(`list`) A list of combined fields generated by
[`.tib_combine()`](https://tibblify.wrangle.zone/dev/reference/dot-tib_combine.md).
