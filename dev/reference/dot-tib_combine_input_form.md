# Combine input forms from a list of tibs

Combine input forms from a list of tibs

## Usage

``` r
.tib_combine_input_form(tib_list, .call)
```

## Arguments

- tib_list:

  (`list`) A list of tib fields.

- .call:

  (`environment`) The environment to use for error messages.

## Value

(`character(1)`) The combined input form.
