# Combine types from a list of tibs

Combine types from a list of tibs

## Usage

``` r
.tib_combine_type(tib_list, .call)
```

## Arguments

- tib_list:

  (`list`) A list of tib fields.

- .call:

  (`environment`) The environment to use for error messages.

## Value

(`character(1)`) The combined tib type.
