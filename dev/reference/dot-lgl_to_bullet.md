# Convert a logical vector to cli bullet symbols

Convert a logical vector to cli bullet symbols

## Usage

``` r
.lgl_to_bullet(x)
```

## Arguments

- x:

  (`logical`) A logical vector where `TRUE` maps to a check mark bullet
  and `FALSE` to a cross bullet.

## Value

(`character`) A character vector of cli bullet names (`"v"` or `"x"`)
the same length as `x`.
